FeiGe 0.4.0 Windows 绿色版

一、启动
1. 将压缩包完整解压到本地文件夹。
2. 双击 FeiGe.exe，无需安装。
3. 项目、设置与密钥保存在程序旁的 FeiGeData 文件夹。

二、首次设置
1. 点击左下角“设置 / Settings”。
2. 选择 OpenAI、Claude、Gemini、DeepSeek、通义千问，或添加 OpenAI 兼容接口。
3. 填写 Base URL、模型名称与 API Key，然后保存并测试连接。
4. API Key 使用 Windows 本机安全存储；项目分享包不会携带密钥。

三、拆镜
1. 新建拆片项目并选择本地视频。
2. 默认使用“混合检测”，融合硬切、白闪、渐隐渐现和叠化检测。
3. 纯硬切素材也可以切换为“经典帧差检测”。
4. 设置中可调整硬切最低帧差、相对倍率、最小切点间隔、最大分镜数和 AI 并发数。
5. 差异曲线单击可定位视频，右键可以在对应时间增加切点。

四、编辑与识别
1. 镜头卡片按钮或右键菜单可以编辑、重新识别和删除镜头。
2. 删除后会自动重新排列镜号并补齐相邻时间区间。
3. 镜头正在识别时，编辑与删除会自动锁定。
4. 可识别全部镜头、重试失败镜头或只重新识别单镜。

五、剧本与版本
1. 点击“AI 生成剧本”；每次生成都会保存为新版本，并记录接口、模型与时间。
2. 在剧本页顶部切换或删除版本，也可以点击“重新生成”比较不同接口的结果。
3. 场景卡片按钮或右键菜单支持编辑、在后面插入和删除。
4. “全文编辑”可以直接修改完整 Markdown；保存内容会同步到当前版本。
5. 切换版本或重新生成时，正在打开的编辑弹窗会自动关闭，防止写错版本。

六、导出
“分别导出”生成 Word、Excel 或 HTML；“导出全部”生成全部成品和分享 ZIP。

七、界面语言
点击左下角“中 / EN”即时切换中文或 English。项目名、AI 内容和导出正文不会被自动翻译。

八、0.4.0 数据说明
0.4.0 使用新的项目结构。首次启动会清空旧项目工作区，但会保留本机模型设置和 API Key。
